# Samples

Archivos de referencia para documentar la estructura de los payloads que SAP
envía a este servicio.

- `admision_legacy.xml` — formato XML original que SAP PI/PO depositaba en el
  drop-folder (referencia histórica, ya no se usa).
- Los formatos JSON esperados están documentados en el README principal del
  proyecto y en los schemas de Swagger (`/docs`).

Estos archivos **no son parte del runtime** del servicio. Existen únicamente
como documentación.
