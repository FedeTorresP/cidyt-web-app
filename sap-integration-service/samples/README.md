# Samples

Archivos XML de referencia para documentar la estructura que SAP PI/PO
deposita en el drop-folder.

Estos archivos **no son parte del runtime** del servicio. Existen únicamente
como documentación del formato esperado y para validación del parser durante
desarrollo.

En producción, SAP PI/PO es quien deposita los XML en su carpeta de salida
(el drop-folder del servidor). Nosotros solo montamos ese volumen en el
contenedor.
